# 📤 رفع الملفات يدوياً على GitHub (بدون Git)

## لو مش قادر تشغل Git، دي أسهل طريقة:

### الخطوة 1: اعمل Repo على GitHub
1. روح [github.com/new](https://github.com/new)
2. اسم المشروع: `golden-coffee-erp`
3. **متعلّش** ✓ Add a README
4. اضغط **Create repository**

### الخطوة 2: ارفع الملفات
1. في صفحة الـ Repo الجديد، هتلاقي:
   ```
   ...or drag and drop files here
   ```
2. اضغط على **"uploading an existing file"**
3. افتح مجلد `GoldenCoffee_ERP_Project` في File Explorer
4. اختار **كل الملفات** (Ctrl+A)
5. اسحبهم (Drag & Drop) في صفحة GitHub
6. اكتب في Commit: `Initial commit`
7. اضغط **Commit changes**

### الخطوة 3: تأكد إن الملفات اترفعت
- افتح الـ Repo على GitHub
- لازم تشوف `package.json`, `prisma/`, `app/`, `components/`

### الخطوة 4: انشر على Render
اتبع [DEPLOY-RENDER.md](DEPLOY-RENDER.md)

## 💡 ملاحظة
الرفع اليدوي ممكن ياخد شوية وقت (حسب سرعة النت)، بس هو أسهل طريقة لو مش عارف تستخدم Git.
